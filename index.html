Grupo Salinas

Collections KPI Dashboard — Tableau & AWS S3

Purpose: Designed and deployed a Tableau dashboard connected to AWS S3 to monitor collections KPIs at the agent and team level, enabling continuous performance tracking and data-driven operational decisions.

Tech Stack

Tableau Desktop · AWS S3 · Python (data preparation) · CSV / Parquet pipelines

Data Architecture
  • Raw collections data ingested weekly into AWS S3 buckets organized by date partition
  • Python scripts used to clean, transform and export structured files ready for Tableau
  • Tableau connected directly to S3 as a live data source, eliminating manual file handling
KPIs Monitored
  • Deficit: gap between collected amount and weekly quota per agent
  • Collections per agent: individual productivity and team ranking
  • Weekly quota achievement: % alcance vs target with week-over-week trend
  • Cumulative deficit: rolling gap tracker to anticipate end-of-month shortfalls
  • Portfolio breakdown: performance segmented by subproject (Italika, Hipotecario, etc.)
Impact

Enabled team leaders to identify underperforming agents earlier in the week, improving intervention timing and supporting a more data-driven collections strategy across the legal portfolio.
